**사용법**

1: 반복법을 사용한 n!(팩토리얼)의 계산

2: 재귀법을 이용한 n!(팩토리얼)의 계산

3: 두 방식 모두 사용 후 결과 / 시간 비교

4: 준비된 테스트 데이터(전역 변수 TEST) 일괄 계산

q: 프로그램 종료(Q도 가능)


**테스트 데이터**

[0, 1, 2, 3, 5, 10, 15, 20, 30, 50, 100]
